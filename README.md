# Git
**git branch [branch name]** : créer une branche, ex: git branch Flo
**git checkout [branch name]** : se positioner sur une branche en local, ex: git checkout main
**git push --set-upstream origin [branch name]** : pousser la branche sur le répo distant
**git pull** : récupérer les modifications distantes
**git add [dossier ou fichier]** : ajouter les dossiers/fichiers au prochain commit, ex: git add .
**git commit -m "message"** : créer un commit avec les fichiers ajoutés
**git push** : pousser le commit sur le répo distant
**git merge [branch name]**: fusionner [branch name] dans la branche courante, ex: git merge main (en étant sur une autre branche)
